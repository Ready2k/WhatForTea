A — Contextual Header
Smallest change. Adds greeting + date. This Week tile shows real date range. Everything else stays put.
B — Today First
Tonight's planned meal is the hero. Best for daily use once plans are set. Requires tonight's meal to be in the planner.
C — Full Width Hero
Cook Now dominates as a wide hero. Secondary tiles shrink to 3-col. Clean visual hierarchy, bold and modern.
D — Glanceable Dashboard
Same grid, much richer data. Cook Now spans full width with top recipes listed inline. Week dots show planned days. Pantry health bar.
E — Smart Hybrid
Dynamic UI. Shows Hero card when a meal is planned, falls back to rich Dashboard (Option D) when not.
